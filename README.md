# Robot Profile Viewer

A responsive single-page application built as part of my frontend development learning journey.

This project was created to explore API consumption, asynchronous JavaScript, and modern frontend tooling. The application fetches robot profile data from a public API and presents it through an interactive and responsive user interface.

It also marked several important milestones in my growth as a developer, including my first experience with Vite, the Fetch API, GitHub Actions workflows, and automated deployment using GitHub Pages.
